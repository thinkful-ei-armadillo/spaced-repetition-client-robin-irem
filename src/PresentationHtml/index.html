Spaced Repetition: SpadesSpanish -- ♠Spanish --

Project overview

♠Spanish is a fullstack app, optimized with computer science.

♠Spanish helps users learn Spanish easily with spaced repetition learning technique.

Live | Client Repo | Server Repo

How Does it work?

The app requires a user to register for an account. Afterwards, when they sign up, the user accesses the dashboard where a list of words appear.

The users can see their progress by skimming the dashboard quickly, taking all the correct and incorrect past practices

If they choose to start practicing, the app would simply provide whereever they left with their practice.

The algorithm would give them the needed word for the correct implementation of spaced repetition technique.

The most challenging part of learning a language is repetition and retaining new vocabulary.

♠Spanish helps users pratice without flashcards and in the way that they will retain their newly acquired vocabulary for a lifetime!

Why Us

♠Spanish makes use of the spaced repetition techniques, known for its positive outcomes especially learning a large number of new items, such as vocabulary.

It implements the technique by using a data structure (singly linked list).

User can concentrate solely on words because of ♠Spanish's simple design and soft colors!

Technologies Used:

  • Front-end: React (Router, Context), Cypress
  • Back-end: Node.js, Express, Postgres, Mocha/Chai
  • Deployment: NOW/Zeit, Heroku
  • Development: Git, GitHub
